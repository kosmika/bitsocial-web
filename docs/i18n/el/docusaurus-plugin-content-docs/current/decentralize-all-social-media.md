# Αποκέντρωση όλων των μέσων κοινωνικής δικτύωσης

Αυτή είναι η τελική κατάσταση που περιγράφεται από το γενικό σχέδιο Bitsocial: όχι μία εφαρμογή, αλλά ένα οικοσύστημα εφαρμογών Bitsocial που μπορεί να ανταγωνιστεί κάθε μεγάλη κατηγορία κοινωνικών μέσων.

## Το μοντέλο

Στην προτεινόμενη ώριμη κατάσταση του δικτύου:

- πολλοί δημόσιοι πάροχοι RPC ανταγωνίζονται για να φιλοξενήσουν κόμβους προφίλ
- Οι χρήστες μπορούν ακόμα να φιλοξενήσουν από μόνοι τους
- Οι αλγόριθμοι τροφοδοσίας γίνονται προαιρετικές υπηρεσίες αντί του υποχρεωτικού νόμου της πλατφόρμας
- Η δημιουργία εσόδων δεν εξαρτάται από το ότι οι τράπεζες είναι πρόθυμες να διατηρήσουν την πλατφόρμα ζωντανή
- Οι εφαρμογές ανταγωνίζονται την ποιότητα των προϊόντων, την ανακάλυψη και τη σχεδίαση διεπαφής

## Πώς θα έμοιαζε η επιτυχία

Οι Bitsocial εφαρμογές θα πρέπει να μπορούν να αμφισβητούν τις πλατφόρμες σε διάφορες μορφές, όχι μόνο σε μία θέση:

- κείμενο και συζήτηση
- πίνακες εικόνας
- φόρουμ
- κοινότητες δημιουργών
- βίντεο σύντομης μορφής και ευρύτερα προϊόντα πολυμέσων

Ο στόχος δεν είναι να επιβάλουμε μια καθολική διεπαφή σε όλους. Είναι να δώσει σε πολλές εφαρμογές πρόσβαση στις ίδιες υποκείμενες κοινότητες, ταυτότητες και δίκτυο.

## Γιατί το επίπεδο RPC έχει σημασία εδώ

Οι πάντα ανοιχτές κοινότητες και τα προφίλ γίνονται πολύ πιο πρακτικά όταν υπάρχουν δημόσιες αγορές RPC. Χωρίς αυτό το επίπεδο, η αυτο-φιλοξενία παραμένει πολύ υψηλή για πολλούς χρήστες, ειδικά σε κινητά.

## Γιατί το επίπεδο δικτύου έχει σημασία εδώ

Εάν η δημιουργία εσόδων από περιεχόμενο μπορεί να διακοπεί από τη συνήθη πίεση τραπεζών ή πλατφόρμας, τότε το οικοσύστημα παραμένει εύθραυστο. Το Bitsocial Chain προτείνεται ως το οικονομικό επίπεδο που κάνει τον μακροπρόθεσμο ανταγωνισμό πιο αξιόπιστο.

## Το βασικό στοίχημα

Το μεγάλο στοίχημα είναι ότι τα μέσα κοινωνικής δικτύωσης μπορούν να χωριστούν σε επίπεδα που μπορούν να αντικατασταθούν:

- πρωτόκολλο
- κοινότητες
- ταυτότητα
- φιλοξενία
- ανακάλυψη
- δημιουργία εσόδων
- εφαρμογές

Μόλις αυτά τα επίπεδα αντικατασταθούν, καμία μεμονωμένη εταιρεία δεν χρειάζεται να κατέχει ολόκληρο το κοινωνικό γράφημα για να λειτουργήσει το προϊόν.
