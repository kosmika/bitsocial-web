# Τοπική μετριοπάθεια, όχι παγκόσμιες απαγορεύσεις

Το Bitsocial δεν είναι απαλλαγμένο από μέτρο. Είναι μετριοπάθεια χωρίς υπερ-διαχειριστή σε επίπεδο πρωτοκόλλου.

Οι ιδιοκτήτες της κοινότητας εξακολουθούν να θέτουν κανόνες για τους δικούς τους χώρους. Οι εφαρμογές εξακολουθούν να αποφασίζουν τι ευρετηριάζουν, κατατάσσουν ή επισημαίνουν. Αυτό που δεν περιλαμβάνει το Bitsocial είναι μια παγκόσμια αρχή που μπορεί να κατασχέσει μια κοινότητα ή να διαγράψει μια ταυτότητα από το ίδιο το δίκτυο.

Για μια ευρύτερη απλή αγγλική περιγραφή της ιδιοκτησίας της κοινότητας, της δημοσίευσης, των προκλήσεων και των υπογραφών επόπτη, διαβάστε [Μια πλήρης απλή εξήγηση του πρωτοκόλλου Bitsocial](./layman-protocol-explanation.md).

## Πώς λειτουργεί το μέτρο

- Ένας κάτοχος κοινότητας μπορεί να εποπτεύσει αυτήν την κοινότητα
- Μια εφαρμογή μπορεί να επιλέξει τι ευρετηριάζει ή τι εμφανίζει
- Ένας χρήστης μπορεί να μετακινηθεί σε άλλη κοινότητα ή άλλη εφαρμογή

Εάν μια εφαρμογή σας φιλτράρει, μια άλλη εφαρμογή μπορεί να επιλέξει να σας εμφανίσει. Εάν μια κοινότητα σας απορρίψει, μια άλλη κοινότητα μπορεί να σας αποδεχτεί ή μπορείτε να δημιουργήσετε τη δική σας.

## Τι αποφεύγει η Bitsocial

Το Bitsocial αποφεύγει το γνωστό σημείο πνιγμού της πλατφόρμας όπου μια εταιρεία ελέγχει:

- το κοινωνικό γράφημα
- τη βάση δεδομένων εποπτείας
- την επιφάνεια ανακάλυψης
- τον χώρο ονομάτων του λογαριασμού

Αυτό το κεντρικό σημείο είναι που μετατρέπει μια απόφαση πολιτικής προϊόντος σε απαγόρευση σε όλο το δίκτυο.

## Το πρακτικό συμβιβασμό

Η τοπική εποπτεία δεν σημαίνει ότι κάθε εφαρμογή θα μοιάζει πανομοιότυπη. Σημαίνει ότι οι συνέπειες της μετριοπάθειας παραμένουν περιορισμένες:

- οι κοινοτικοί κανόνες ισχύουν εντός αυτής της κοινότητας
- Οι κανόνες εφαρμογής ισχύουν μέσα σε αυτήν την εφαρμογή
- ούτε γίνεται αυτόματα νόμος πρωτοκόλλου

Αυτό προσπαθεί να διατηρήσει το «χωρίς παγκόσμιες απαγορεύσεις».
