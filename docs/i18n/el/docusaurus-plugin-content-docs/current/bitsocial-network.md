# Bitsocial Chain

Το Bitsocial Chain είναι το όνομα που χρησιμοποιείται στο κύριο σχέδιο για το κοινό επίπεδο αλυσίδας εφαρμογών που θα βρίσκεται κάτω από τις εφαρμογές Bitsocial.

Η ιδέα δεν είναι να αντικαταστήσουμε το peer-to-peer κοινωνικό στρώμα. Είναι να παρέχει τα κοινά οικονομικά και συντονιστικά πρωτόγονα που θα χρειαστούν οι κοινωνικές εφαρμογές, εάν πρόκειται να ανταγωνιστούν τις κατεστημένες πλατφόρμες σε κλίμακα.

## Τι σημαίνει η εξουσία

- αποκεντρωμένους τομείς Bitsocial όπως το `.bso`
- κοινή ρευστότητα σε όλες τις εφαρμογές
- ανθεκτικές ράγες δημιουργίας εσόδων
- χρηματοπιστωτικές δομές που είναι πιο δύσκολο για τις τράπεζες ή τις πλατφόρμες να πνιγούν
- εφέ δικτύου που δεν εξαρτώνται από μια εταιρεία που κατέχει ολόκληρη τη στοίβα

## Γιατί έχει σημασία

Η αποκέντρωση των κοινοτήτων και των ταυτοτήτων είναι απαραίτητη, αλλά δεν αρκεί η αποκέντρωση όλων των μέσων κοινωνικής δικτύωσης.

Εάν το οικονομικό επίπεδο εξακολουθεί να εξαρτάται από μια χούφτα κεντρικών σιδηροδρομικών γραμμών, τότε οι εφαρμογές παραμένουν εύκολο να πιεστούν, να αποπλαστοποιηθούν ή να λιμοκτονήσουν οικονομικά. Το Bitsocial Chain είναι η προτεινόμενη απάντηση σε αυτό το δεύτερο επίπεδο εξάρτησης.

## Σχέση με το υπόλοιπο Bitsocial

Το Bitsocial Chain προτείνεται ως κοινόχρηστη υποδομή για εφαρμογές Bitsocial, όχι ως το προϊόν με το οποίο αλληλεπιδρούν απευθείας οι χρήστες καθημερινά.

Το αποτέλεσμα στο κοινό θα ήταν:

- οι κοινότητες παραμένουν ομότιμες
- οι εφαρμογές παραμένουν διαφοροποιημένες
- η δημιουργία εσόδων και η ονομασία γίνονται πιο ανθεκτικές
- Η αξία μπορεί να μετακινηθεί σε όλο το οικοσύστημα χωρίς να δημιουργήσει εκ νέου έναν κεντρικό ιδιοκτήτη πλατφόρμας

## Γιατί αυτό βρίσκεται στη Φάση 4

Το κύριο σχέδιο το τοποθετεί μετά από τα προηγούμενα σημεία απόδειξης σε επίπεδο εφαρμογής:

1. αποκέντρωση των πινάκων εικόνας
2. εκκινήστε το Bitsocial RPC
3. αποκέντρωση φόρουμ
4. εκκινήστε το Bitsocial Chain

Αυτή η ακολουθία έχει νόημα επειδή ένα επίπεδο συντονισμού σε επίπεδο οικοσυστήματος είναι πιο χρήσιμο όταν υπάρχουν ήδη πραγματικές εφαρμογές και πραγματική συμπεριφορά χρήστη στο δίκτυο.

## Αρχές σχεδιασμού

Επειδή το Bitsocial Chain παραμένει προτεινόμενη υποδομή και όχι ήδη λανσαρισμένο προϊόν, το τρέχον σχέδιο πρέπει να παραμείνει πειθαρχημένο σε μερικά σημεία:

- Πρώτα οι εφαρμογές και οι κοινότητες. Το επίπεδο του δικτύου πρέπει να ακολουθεί την πραγματική χρήση μέσα στις εφαρμογές Bitsocial και όχι να προηγείται με κερδοσκοπικά οικονομικά χαρακτηριστικά.
- Πραγματική συνεισφορά αντί για hype. Αν το Bitsocial προσθέσει αργότερα περιουσιακά στοιχεία σε επίπεδο κοινότητας, δημιουργού ή client, αυτά πρέπει να ανταμείβουν τη συμμετοχή και το μακροπρόθεσμο χτίσιμο αντί για αντιγραφές launch ή βραχυπρόθεσμο trading.
- Επιτρέπεται η επιμέλεια. Οι εφαρμογές Bitsocial μπορούν να διαμορφώνουν κατατάξεις, προεπιλογές, ανακάλυψη και χρονισμό διάθεσης ώστε να ευνοούν ανθεκτικές κοινότητες αντί για συμπεριφορά τύπου καζίνο.
- Η σταδιακή διάθεση είναι υγιής. Είναι λογικό να καθυστερήσουν τα χρηματοοικονομικά χαρακτηριστικά που είναι εγγενή στο appchain μέχρι το οικοσύστημα να έχει αρκετούς χρήστες, εργαλεία moderation και σαφήνεια προϊόντος για να τα υποστηρίξει με ασφάλεια.
- Οι ακριβείς μηχανισμοί παραμένουν ανοιχτοί. Αυτή η σελίδα εξηγεί τον ρόλο του Bitsocial Chain, όχι μια κλειδωμένη υπόσχεση για token launchers, φόρμουλες τελών ή τα τελικά tokenomics.
