# Bitsocial Chain

Bitsocial Chain เป็นชื่อที่ใช้ในแผนหลักสำหรับเลเยอร์ appchain ที่ใช้ร่วมกันซึ่งจะอยู่ภายใต้แอป Bitsocial

แนวคิดนี้ไม่ใช่การแทนที่เลเยอร์ทางสังคมแบบเพียร์ทูเพียร์ มันคือการจัดหาพื้นฐานทางเศรษฐกิจที่ใช้ร่วมกันและการประสานงานที่แอปโซเชียลจำเป็นต้องใช้หากพวกเขาจะแข่งขันกับแพลตฟอร์มที่มีหน้าเดิมในวงกว้าง

## อำนาจหมายถึงอะไร

- โดเมน Bitsocial แบบกระจายอำนาจ เช่น `.bso`
- แบ่งปันสภาพคล่องระหว่างแอพ
- รางการสร้างรายได้ที่ทนทาน
- โครงสร้างทางการเงินที่ยากสำหรับธนาคารหรือแพลตฟอร์มที่จะปิดกั้น
- เอฟเฟกต์เครือข่ายที่ไม่ได้ขึ้นอยู่กับบริษัทเดียวที่เป็นเจ้าของสแต็กทั้งหมด

## ทำไมมันถึงสำคัญ

การกระจายอำนาจให้กับชุมชนและอัตลักษณ์เป็นสิ่งที่จำเป็น แต่ยังไม่เพียงพอในการกระจายอำนาจของโซเชียลมีเดียทั้งหมด

หากชั้นทางเศรษฐกิจยังคงขึ้นอยู่กับรางที่รวมศูนย์จำนวนหนึ่ง แอปต่างๆ ก็ยังคงง่ายต่อการกดดัน เลิกใช้แพลตฟอร์ม หรือขาดแคลนทางการเงิน Bitsocial Chain เป็นคำตอบที่นำเสนอสำหรับการพึ่งพาชั้นที่สองนั้น

## ความสัมพันธ์กับส่วนที่เหลือของ Bitsocial

Bitsocial Chain ได้รับการเสนอให้เป็นโครงสร้างพื้นฐานที่ใช้ร่วมกันสำหรับแอป Bitsocial ไม่ใช่ผลิตภัณฑ์ที่ผู้ใช้โต้ตอบด้วยโดยตรงทุกวัน

ผลลัพธ์ที่เปิดเผยต่อสาธารณะจะเป็น:

- ชุมชนอยู่แบบเพียร์ทูเพียร์
- แอพยังคงความแตกต่าง
- การสร้างรายได้และการตั้งชื่อมีความคงทนมากขึ้น
- มูลค่าสามารถเคลื่อนข้ามระบบนิเวศได้โดยไม่ต้องสร้างเจ้าของแพลตฟอร์มแบบรวมศูนย์ขึ้นมาใหม่

## เหตุใดจึงอยู่ในระยะที่ 4

แผนหลักวางสิ่งนี้ไว้หลังจุดพิสูจน์ระดับแอปก่อนหน้านี้:

1. กระจายอำนาจบอร์ดภาพ
2. เปิดตัว Bitsocial RPC
3. กระจายอำนาจฟอรัม
4. เปิดตัว Bitsocial Chain

ลำดับนั้นสมเหตุสมผลเพราะเลเยอร์การประสานงานระดับระบบนิเวศจะมีประโยชน์มากกว่าเมื่อแอปจริงและพฤติกรรมผู้ใช้จริงมีอยู่แล้วบนเครือข่าย

## หลักการออกแบบ

เนื่องจาก Bitsocial Chain ยังเป็นโครงสร้างพื้นฐานที่เสนอไว้ ไม่ใช่ผลิตภัณฑ์ที่เปิดใช้งานแล้ว แผนปัจจุบันจึงควรรักษาวินัยไว้ในบางด้าน:

- แอปและชุมชนต้องมาก่อน เลเยอร์เครือข่ายควรตามการใช้งานจริงในแอป Bitsocial ไม่ใช่นำหน้าไปด้วยฟีเจอร์การเงินเชิงเก็งกำไร
- คุณค่าจากการมีส่วนร่วมจริงสำคัญกว่ากระแส hype หากต่อไป Bitsocial เพิ่มสินทรัพย์ระดับชุมชน ผู้สร้าง หรือไคลเอนต์ สินทรัพย์เหล่านั้นควรให้รางวัลกับการมีส่วนร่วมและการสร้างระยะยาว แทนการเปิดตัวเลียนแบบหรือการเทรดระยะสั้น
- อนุญาตให้มีการคัดสรรได้ แอป Bitsocial สามารถกำหนดการจัดอันดับ ค่าตั้งต้น การค้นพบ และจังหวะการเปิดใช้ เพื่อให้ความสำคัญกับชุมชนที่ยั่งยืนมากกว่าพฤติกรรมแบบคาสิโน
- การเปิดใช้แบบค่อยเป็นค่อยไปเป็นสิ่งที่ดี สมเหตุสมผลที่จะเลื่อนฟีเจอร์การเงินแบบ native ของ appchain ออกไปจนกว่าระบบนิเวศจะมีผู้ใช้ เครื่องมือ moderation และความชัดเจนของผลิตภัณฑ์เพียงพอที่จะรองรับได้อย่างปลอดภัย
- กลไกที่แน่นอนยังเปิดอยู่ หน้านี้อธิบายบทบาทของ Bitsocial Chain ไม่ใช่คำสัญญาที่ล็อกไว้เกี่ยวกับ token launcher สูตรค่าธรรมเนียม หรือ tokenomics ขั้นสุดท้าย
