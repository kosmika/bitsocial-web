# การกลั่นกรองในท้องถิ่น ไม่ใช่การแบนทั่วโลก

Bitsocial ไม่ได้ปราศจากการกลั่นกรอง เป็นการกลั่นกรองโดยไม่มีผู้ดูแลระบบขั้นสูงระดับโปรโตคอล

เจ้าของชุมชนยังคงตั้งกฎเกณฑ์สำหรับพื้นที่ของตนเอง แอปยังคงตัดสินใจว่าจะจัดทำดัชนี จัดอันดับ หรือไฮไลต์อะไร สิ่งที่ Bitsocial ไม่ได้รวมไว้คือหน่วยงานระดับโลกที่สามารถยึดชุมชนหรือลบข้อมูลประจำตัวออกจากเครือข่ายได้

สำหรับคำแนะนำแบบอังกฤษธรรมดาที่กว้างขึ้นเกี่ยวกับการเป็นเจ้าของชุมชน การเผยแพร่ การท้าทาย และลายเซ็นของผู้ดำเนินรายการ โปรดอ่าน [คำอธิบายทั่วไปที่สมบูรณ์ของโปรโตคอล Bitsocial](./layman-protocol-explanation.md).

## การกลั่นกรองทำงานอย่างไร

- เจ้าของชุมชนสามารถกลั่นกรองชุมชนนั้นได้
- แอปสามารถเลือกสิ่งที่จะจัดทำดัชนีหรือแสดงได้
- ผู้ใช้สามารถย้ายไปยังชุมชนอื่นหรือแอปอื่นได้

หากแอปหนึ่งกรองคุณ แอปอื่นก็ยังสามารถเลือกที่จะแสดงให้คุณเห็นได้ หากชุมชนหนึ่งปฏิเสธคุณ ชุมชนอื่นก็สามารถยอมรับคุณได้ หรือคุณสามารถสร้างชุมชนของคุณเองก็ได้

## สิ่งที่ Bitsocial หลีกเลี่ยง

Bitsocial หลีกเลี่ยงจุดปิดกั้นแพลตฟอร์มที่คุ้นเคยซึ่งบริษัทหนึ่งควบคุม:

- กราฟทางสังคม
- ฐานข้อมูลการกลั่นกรอง
- พื้นผิวการค้นพบ
- เนมสเปซของบัญชี

จุดศูนย์กลางคือสิ่งที่เปลี่ยนการตัดสินใจนโยบายผลิตภัณฑ์ให้กลายเป็นการห้ามทั่วทั้งเครือข่าย

## การแลกเปลี่ยนในทางปฏิบัติ

การกลั่นกรองเฉพาะที่ไม่ได้หมายความว่าทุกแอปจะมีหน้าตาเหมือนกัน หมายความว่าผลของการกลั่นกรองยังคงมีขอบเขต:

- กฎของชุมชนมีผลใช้ภายในชุมชนนั้น
- กฎของแอปมีผลภายในแอปนั้น
- ไม่กลายเป็นกฎหมายโปรโตคอลโดยอัตโนมัติ

นั่นคือสิ่งที่ “ไม่มีการแบนทั่วโลก” พยายามรักษาไว้
