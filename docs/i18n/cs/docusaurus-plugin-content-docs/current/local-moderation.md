# Místní moderování, nikoli globální zákazy

Bitsocial není bez moderování. Jedná se o moderování bez supersprávce na úrovni protokolu.

Majitelé komunit si stále stanovují pravidla pro své vlastní prostory. Aplikace stále rozhodují o tom, co indexují, hodnotí nebo zvýrazní. Co Bitsocial nezahrnuje, je globální autorita, která může zabavit komunitu nebo vymazat identitu ze samotné sítě.

Chcete-li získat širší jednoduchý anglický návod na vlastnictví komunity, publikování, výzvy a podpisy moderátorů, přečtěte si [Kompletní laické vysvětlení protokolu Bitsocial](./layman-protocol-explanation.md).

## Jak funguje moderování

- Vlastník komunity může tuto komunitu moderovat
- Aplikace si může vybrat, co bude indexovat nebo zobrazovat
- Uživatel se může přesunout do jiné komunity nebo jiné aplikace

Pokud vás jedna aplikace filtruje, jiná aplikace se stále může rozhodnout, že vám ji zobrazí. Pokud vás jedna komunita odmítne, může vás přijmout jiná komunita nebo si můžete vytvořit vlastní.

## Čemu se Bitsocial vyhýbá

Bitsocial se vyhýbá známému škrtícímu bodu platformy, kde jedna společnost ovládá:

- sociální graf
- databázi moderování
- povrch objevu
- jmenný prostor účtu

Ústředním bodem je to, co mění rozhodnutí o produktové politice v celosíťový zákaz.

## Praktický kompromis

Místní moderování neznamená, že každá aplikace bude vypadat stejně. To znamená, že důsledky umírněnosti zůstávají v rozsahu:

- uvnitř této komunity platí pravidla komunity
- uvnitř této aplikace platí pravidla aplikace
- ani jedno se automaticky nestává protokolárním zákonem

To je to, co se „žádné globální zákazy“ snaží zachovat.
